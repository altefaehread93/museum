# Egyptian Papyrus Exhibit - Team Role Quest

Interactive museum questionnaire application inspired by ancient Egyptian papyrus and wisdom.

## Quick Start

### Frontend Only (Recommended)

```bash
# Install dependencies
npm install

# Start development server
npm run dev
```

Visit http://localhost:3000

### Full Stack (Frontend + Backend)

1. **Install frontend dependencies**
   ```bash
   npm install
   ```

2. **Install backend dependencies**
   ```bash
   cd backend
   npm install
   cd ..
   ```

3. **Start backend server** (in one terminal)
   ```bash
   # Windows
   scripts\start-backend.bat
   
   # Linux/Mac
   bash scripts/start-backend.sh
   ```

4. **Start frontend dev server** (in another terminal)
   ```bash
   # Windows
   scripts\start-dev.bat
   
   # Or directly
   npm run dev
   ```
